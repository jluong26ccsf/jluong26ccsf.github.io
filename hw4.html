List Descriptions

Lists Usage
Unordered List The tag "<ul>" is used create a unordered list. This tag uses bullet points to list things without order or ranking.
An example is listing not the steps, but just the ingredients of a recipe.
Ordered List The tag "<ol>" is to create a ordered list. This tag is uses numbers to list things in order or in ranking.
An example is listing the steps of a recipe or listing things from most favorite to least favorite.
Definition List The tag "<dl>" is to create a definition list. This tag is used to provide a definition or description of a word/text.
An example is to have the word "sprint" as the term (<dt>) and give a definition (<dd>) of "to run very fast".

Unordered List Example

  • Flour
  • Milk
  • Eggs
  • Salt

Ordered List Example

  1. Crack the eggs to a bowl and mix
  2. Add flour to bowl and mix
  3. Pour milk to bowl and mix
  4. Add a teaspoon of salt
  5. Finish
